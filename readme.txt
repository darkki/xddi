DriveType Data type: uint32 https://docs.microsoft.com/en-us/windows/win32/cimwin32prov/win32-logicaldisk
Numeric value that corresponds to the type of disk drive this logical disk represents.

Unknown (0)
No Root Directory (1)
Removable Disk (2)
Local Disk (3)
Network Drive (4)
Compact Disc (5)
RAM Disk (6)
